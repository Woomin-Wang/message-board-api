# Message Board Api Server
Spring Boot를 이용해 개발한 간단한 메시지 보드 REST API 서버입니다.

<br>

### Tech Stack
- Java 21
- Spring Boot 3.5.6
- Spring Data JPA
- Gradle
- MySQL / H2
